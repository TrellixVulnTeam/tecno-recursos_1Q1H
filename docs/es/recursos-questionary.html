Questionary¶

Questionary son cuestionarios de tecnología de tipo test con múltiples respuestas de las que solo una es válida.

Los test solo avanzan a la siguiente pregunta cuando se pulsa la respuesta correcta, es decir que tienen realimentación inmediata para poder entrenarse fácilmente.
